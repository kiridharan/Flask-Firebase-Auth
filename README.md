# Flask-Firebase-Auth


# How to start

1. fork this repository and clone it to your local machine.
2. create a branch and contribute to it.

```
$ virtualenv env 
```
- run the package install
```
$ pip install -r requirements.txt
```
windows users:
```
$ env/bin/activate  
```

linux\Mac users:

```
$ source env/bin/activate
```
